# CHANGELOG — Generador de Submittals ES v2.6.10

Fecha: 23 de julio de 2026

## Varias marcas en un mismo material

- Cuando una carpeta contiene fichas técnicas de dos o más fabricantes
  distintos, el campo **Marca** muestra todas las alternativas detectadas.
- El formato es `Marca 1 / Marca 2 / Marca 3`.
- Se eliminan marcas repetidas, variantes casi idénticas y valores genéricos
  como `SIN ESPECIFICAR` o `POR DEFINIR`.
- El orden respeta la aparición de las marcas en las fichas técnicas.
